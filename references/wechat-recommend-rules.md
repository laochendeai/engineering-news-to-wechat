# 微信公众号推荐机制优化指南

基于微信公众号"看一看"、"推荐"、"发现页"等推荐流量入口的规则，整理的写作规范。

## 一、微信推荐的流量入口

1. **看一看（在看）**：读者点击"在看"后，文章出现在其好友的"看一看"信息流
2. **推荐信息流**：微信算法主动推荐，出现在"发现页→看一看→推荐"tab
3. **话题标签推荐**：文章带的话题标签被推荐给关注该话题的用户
4. **搜一搜**：用户主动搜索时的结果排名
5. **阅读原文链路**：被其他公众号引用、群转发后的二次传播

## 二、推荐算法核心指标

微信推荐算法主要看以下指标（按权重排序）：

| 指标 | 权重 | 说明 |
|------|------|------|
| **完读率** | ⭐⭐⭐⭐⭐ | 最核心指标。读者从第一行读到最后一行的比例 |
| **分享率** | ⭐⭐⭐⭐⭐ | 读者转发到朋友圈/群/好友的比例 |
| **在看率** | ⭐⭐⭐⭐ | 读者点击"在看"的比例 |
| **收藏率** | ⭐⭐⭐⭐ | 读者收藏的比例，体现内容"有用性" |
| **阅读停留时长** | ⭐⭐⭐⭐ | 读者在文章上的平均停留时间 |
| **互动率（评论/点赞）** | ⭐⭐⭐ | 读者参与讨论的程度 |
| **点击率（打开率）** | ⭐⭐⭐ | 标题/封面被点击打开的比例 |
| **负反馈率** | ⭐⭐⭐⭐⭐ | 读者投诉/不感兴趣/取关，负反馈过高会降权 |

## 三、写作规则（按推荐友好度排序）

### 规则1：完读率是第一优先级

- **文章长度**：1500-3000字最佳。太短没深度，太长完读率崩
- **开头100字决定生死**：前100字必须让读者觉得"这篇对我有用"
- **段落要短**：每段不超过4-5行（手机屏幕适配）
- **多用小标题**：H2/H3 切割内容，读者可以快速跳读
- **表格和清单**：比纯文字更容易被"扫读"完
- **不要用长引用块、大段代码**：手机上阅读体验差，直接导致跳出

### 规则2：标题决定点击率

- **15-30字最佳**，太短没信息量，太长被截断
- **包含数字**：如"12个关键点""6个常见错误"
- **包含场景/人群**：如"弱电施工队必须掌握"
- **制造好奇心**：但不要标题党（高负反馈）
- **避免**：纯品牌词、过于抽象、无信息量的标题

### 规则3：封面图决定点击率

- **尺寸**：900×383px（2.35:1）最佳，微信公众号推荐流标准尺寸
- **文字少而精**：封面图上最多2行字，字号要大
- **颜色对比**：深色底+亮色字，在浅色列表中跳出来
- **不要**：纯色块无内容、文字过小、图片模糊、内容与标题不符

### 规则4：内容结构要"可扫读"

- **每个H2标题下不超过3个H3**，结构不超过3层
- **关键结论加粗**：读者只看加粗部分也能获得核心信息
- **善用表格**：对比类内容用表格，手机上表格可横向滑动
- **善用清单**：步骤、要点、注意事项用有序/无序列表
- **每500字放一个"锚点"**：小标题、加粗句、表格、清单，让读者找到阅读节奏

### 规则5：收藏价值决定分享价值

- **自查清单**：读者会收藏备用
- **对比表格**：读者会截图转发
- **实操步骤**：读者会收藏当SOP
- **数据/标准**：读者会收藏当参考资料

### 规则6：开头和结尾要有"钩子"

- **开头**：直接点明"这篇能帮你解决什么问题"，不要铺垫
- **结尾**：引导互动——提问、征集、预告下一篇，提升评论率

### 规则7：发布时间影响初始推荐

- **工作日**：7:30-8:30（通勤）、12:00-13:00（午休）、18:00-19:30（下班）
- **周末**：9:00-10:30、20:00-22:00
- **发布后1小时内**的阅读数据最关键，决定是否进入推荐池

### 规则8：话题标签

- 每篇文章最多3-5个话题标签
- 标签要精准匹配目标读者关注的话题
- 弱电行业可用标签：#弱电工程 #智能化 #智慧建筑 #安防监控 #万兆网络 #布线工艺

## 四、负反馈信号（会导致降权）

- 标题党（标题与内容严重不符）
- 广告硬植入
- 低质量转载/洗稿
- 频繁发布低质文章（日更>3篇且质量参差不齐）
- 被大量读者投诉

## 五、从草稿到推荐的路径

```
发布 → 第一批粉丝阅读（1小时内）
     ↓
完读率 + 在看率 + 分享率达标？
     ↓ 是                    ↓ 否
进入推荐池（冷启动）      限流，仅粉丝可见
     ↓
第二批推荐阅读（非粉丝）
     ↓
互动指标持续达标？
     ↓ 是                    ↓ 否
扩大推荐范围              推荐衰减，停止分发
     ↓
进入"看一看→推荐"持续曝光
```

**冷启动门槛**（经验值）：
- 粉丝数 < 1000：完读率 > 40%，在看率 > 5%，即可进入推荐池
- 粉丝数 > 5000：需要更高的绝对互动量

## 六、每日日报类内容的特殊优化

- **固定栏目名**：如"弱电智能化工程圈日报"，形成品牌认知
- **统一格式**：固定结构让读者形成阅读习惯
- **字数控制**：日报 1500-2500字，专题文章 2000-4000字
- **发布时间**：固定时间发布，培养读者习惯（建议工作日 7:30 或 18:00）
